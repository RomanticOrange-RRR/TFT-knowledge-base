## 回答

# TFT Set 17 星の観測者（Stargazer）シナジー解説

## シナジー効果

星の観測者は**ユニット数に応じた段階的なボーナス**を提供します：

| ユニット数 | 効果 |
|-----------|------|
| **2体** | マナ生成量 +15% |
| **4体** | マナ生成量 +15% + 攻撃速度 +20% |
| **6体** | マナ生成量 +15% + 攻撃速度 +20% + スキル威力 +25% |

## 主要チャンピオン

- **ジャンナ**（3コスト）
- **ニーコ**（4コスト）
- **アイバーン**（5コスト）
- **ソラカ**（5コスト）

## 実践的な活用ポイント

### ✅ 強い使い方
1. **マナ駆動型ビルド**と相性良好
   - 複数ユルトやジャンナの頻繁なULT起動が可能
   
2. **4体以上の搭載を狙う**
   - 攻撃速度ボーナスがDPS向上に直結

3. **サポートシナジーとの組み合わせ**
   - 星の観測者単体では火力不足なため、他シナジーとの複合構成必須

### ⚠️ 注意点
- **スタンドアロンで成立しない**（他シナジーへの依存度高い）
- マナボーナスはULT駆動型チャンピオンがいてこそ活躍

Set 17の詳細バージョンはお持ちですか？パッチによって数値が変動しているかもしれません。
